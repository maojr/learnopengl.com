# learnopengl.com
### Platform

macOS

### Dependencies

Install with [brew](https://brew.sh/):

	brew install glm
	brew install glew
	brew install glfw3

### Compile

Example:

	HiWindow.cpp:      g++ HiWindow.cpp -lglfw -framework OpenGL

	HelloTriangle.cpp: g++ HelloTriangle.cpp -lglew -lglfw -framework OpenGL
